# Exam Information

### About The Exam

| Detail            | Info                                       |
| ----------------- | ------------------------------------------ |
| **Format**        | CTF-style, points-based                    |
| **Total Flags**   | 14                                         |
| **Passing Score** | 85 points (12 flags minimum)               |
| **Report**        | Required — use the official template       |
| **Style**         | Large, realistic penetration test scenario |

> 💡 **Tip:** Always include screenshots of captured flags in your exam report.

### Exam Strategy

1. **Start with enumeration** — don't skip this phase
2. **Document everything** as you go — screenshots, commands, outputs
3. **Use the official report template** — it's structured for a reason
4. **You need 12/14 flags** — don't get stuck, move on and come back
5. **Treat it like a real pentest** — methodology matters
